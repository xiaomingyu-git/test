# Vue Development Command

使用Vue开发指南来分析和处理Vue相关的开发请求。这个命令会触发vue-dev-guidelines技能，帮助你：

- 分析Vue开发需求
- 制定实施计划
- 提供最佳实践建议
- 链接到相关资源文档

## 使用方法

输入 `/vue` 后跟你的Vue开发需求，例如：

- `/vue 创建一个用户管理的CRUD页面`
- `/vue 如何实现一个带验证的表单组件`
- `/vue 优化表格性能问题`
- `/vue 集成富文本编辑器`
- `/vue 设计一个响应式布局`

命令会自动应用Think-First开发流程：
1. 🤔 分析需求
2. 🔍 查询最新文档
3. 📋 制定计划
4. 💻 实施建议
5. ✅ 验证清单

## 示例输入

```
/vue 创建一个商品管理页面，需要包含搜索、分页、批量操作功能
```

```
/vue 我需要实现一个用户注册表单，包含邮箱验证、密码强度检查
```

```
/vue 如何优化一个包含大量数据的表格组件的性能
```

让Vue开发指南帮助你构建更好的Vue应用！
